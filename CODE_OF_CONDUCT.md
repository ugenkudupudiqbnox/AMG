# Code of Conduct

## Our Pledge

We are committed to providing a respectful, professional, and harassment-free environment for everyone participating in the AMG project.

This applies to:
- GitHub issues and pull requests
- Discussions and reviews
- Community interactions

---

## Expected Behavior

Participants are expected to:

- Be respectful and constructive
- Focus on technical merit, not individuals
- Accept feedback professionally
- Communicate clearly and patiently
- Assume good intent

---

## Unacceptable Behavior

The following behaviors are not tolerated:

- Harassment, discrimination, or personal attacks
- Dismissive or hostile language
- Trolling or deliberate disruption
- Repeatedly ignoring maintainersâ€™ guidance
- Pressuring maintainers to accept unsafe changes

---

## Scope

This Code of Conduct applies to all project spaces:
- GitHub repositories
- Issue trackers
- Pull requests
- Community discussions

---

## Enforcement

Project maintainers are responsible for enforcing this Code of Conduct.

Consequences may include:
- Warning
- Temporary ban
- Permanent ban

Decisions are made to protect the health of the project.
