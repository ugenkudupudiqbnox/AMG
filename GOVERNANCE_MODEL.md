# Governance Model

## Project Governance
AMG follows a conservative governance model optimized for correctness and trust.

- BDFL (initial phase)
- Public roadmap
- RFCs for breaking changes
- Explicit non-goals documented

## Decision Principles
- Security > Features
- Determinism > Flexibility
- Auditability > Performance

## Change Policy
Breaking changes require:
- Public discussion
- Clear migration path
- Maintainer approval
