# CISO Security Questionnaire (Simulated)

Q: Can agents bypass governance?
A: No. All memory and context access is mediated by AMG.

Q: How do you stop an agent immediately?
A: Via agent disable or global kill switch.

Q: Is agent behavior auditable?
A: Yes. All actions produce immutable audit logs.

Q: How is data retention enforced?
A: Mandatory TTLs with automatic deletion.

Q: Who is accountable for agents?
A: Each agent has an explicit Agent Owner.

Q: Is learning allowed?
A: No. Learning and adaptation are out of scope for V1.
